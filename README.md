## 環境構築
1. リポジトリをクローン:  
   SSHの場合  
    `git clone git@github.com:DIGIT-KITAQ-Flutter/team-digfit.git`  
    HTTPSの場合  
    `git clone git@github.com:DIGIT-KITAQ-Flutter/team-digfit.git`  
2. プロジェクトディレクトリに移動:  
    `cd digfit`  
3. パッケージのインストール:
   `flutter pub get`  
4. プロジェクトの実行:
   `flutter run`
